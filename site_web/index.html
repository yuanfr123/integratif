Analyse de exportations par pays, produit et événement

Sur ce site web vous trouverez des informations concernant le montant total des exportations de quelques pays vers leurs principaux pays partenaires en fonction des produits, ainsi que par événement historique à compter de 2012. L'objectif étant de voir comment un pays arrive dans certains cas à changer de partenaire à partir d'une situation concrète. Cependant, il n'y a pas une analyse du pourquoi le changement de partenaire. Toutefois, en ayant une vue spécifique, les recherches sur un moteur de recherches seront plus faciles à concrétiser.

Les données proviennent de deux sites web principalement. Les données ont été récupérées dans des fichiers Excel, puis transportées dans notre base de données.

Possibles intéractions du site web

Dans le menu d'en haut, choisissez la thématique souhaitée, soit les vues générales, soit les vues spécifiques. Pour les vues générales, vous avez deux types d'indicateurs : les exportations vers les pays les plus importants par produit, et l'évolution des exportations par produit et pays. Pour les vues spécifiques, vous n'avez qu'un choix possible : la comparaison des exportations entre pays par produit. Choisissez pour chaque indicateur le pays exportateur, le produit et, pour certains cas, l'événement historique (il vous affichera automatiquement la période). Ensuite, cliquez sur "Afficher" pour voir apparaître un tableau et un graphique contenant les informations choisies. Attention, il se peut que certains tableaux ne contiennent pas de chiffres. Cela est dû au fait que les données sont celles qui sont obtenues et que parfois, il manque de données dans les sites web dans lesquels on a extrait les données.
