A propos

Tout au long de l'année en M1 MIAGE IM en formation initiale, nous avons été emmenés à créer un site web intéractif avec l'affichage de données, récupérées d'une base de données, qu'on a créée nous-mêmes. Pour se faire, on a dû maquetter notre site web, modéliser notre base de données et s'organiser dans la gestion de ce projet (utilisation de Trello pour les exigences, plusieurs réunions pour regarder l'avancée du projet, etc). Ici en-dessous, vous trouverez les membres de l'équipe du projet.
